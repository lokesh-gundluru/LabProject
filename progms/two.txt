Multinode
Clone the VM ( full clone ) which you have used for single node

rightclick on the VM and choose Manage then clone 
and click on next then choose full clone 
and name the clone as slave1
After clone creation 

In master 
sudo hostnamectl set-hostname master
then restart the master VM ( Must Done ) 

In slave1
sudo hostnamectl set-hostname slave1
and then restart the slave system too

after restart 

check the hostname in both Master and slave1
using the command:
hostname

now note the ip addresses of both master and slave1
using the command : ip a

 In master and slave1
sudo nano /etc/hosts

add this two lines
192.168.200.? master
192.168.200.? slave1

 then in Master

ssh-copy-id slave1

enter password

In Master

nano $HADOOP_HOME/etc/hadoop/workers

in that remove everything 
and add the word  slave1

save the file using Ctrl+X, Y, Enter

in both master and slave1 do the following 

nano $HADOOP_HOME/etc/hadoop/core-site.xml

replace everything in it with the below code
<configuration>
  <property>
    <name>fs.defaultFS</name>
    <value>hdfs://master:9000</value>
  </property>
</configuration>



nano $HADOOP_HOME/etc/hadoop/hdfs-site.xml

in this file also replace everything with below code 

<configuration>
  <property>
    <name>dfs.replication</name>
    <value>2</value>
  </property>

  <property>
    <name>dfs.namenode.name.dir</name>
    <value>file:/usr/local/hadoop/data/namenode</value>
  </property>

  <property>
    <name>dfs.datanode.data.dir</name>
    <value>file:/usr/local/hadoop/data/datanode</value>
  </property>
</configuration>

 IN master

mkdir -p /usr/local/hadoop/data/namenode
mkdir -p /usr/local/hadoop/data/datanode

 In master format again 

hdfs namenode -format

 in master 

start-dfs.sh

 next verify output in both master and slave1 using the command 


jps

 then open browser in master VM

and open this url 
http://master:9870
